Custom Writing Syntax

Detailed and practical reference for human-like scripted typing.

Quick Start

  1. Enable Custom writing syntax in the popup.
  2. Write directives as [[command:payload]].
  3. Click start and let the script drive typing order.

Important: when directives are detected, built-in random typo/whitespace/pause noise is automatically disabled to avoid conflicts.

Core Directives

DirectiveMeaning
[[rev:draft=>final]]Type draft, pause, delete draft, type final.
[[del:text]]Type text then delete it.
[[pause:1200]]Pause exact milliseconds.
[[pause:500-1600]]Pause with random range.
[[choice:A||B||C]]Pick one option randomly.
[[chance:35|text]]Type nested text with probability.
[[repeat:3|text]]Repeat nested text N times.

Behavior Controls

DirectiveMeaning
[[speed:20-45]]Set speed in seconds per 100 chars.
[[speed:default]]Reset speed to session baseline.
[[typo:on/off/default/2-5]]Control typo behavior inside syntax flow.
[[ws:on/off/default/1-3/keep/drop]]Control whitespace noise behavior.
[[fix:off|immediate|delayed|random|default]]Control typo/ws auto-correction mode.
[[fixdelay:300-1800]]Set correction delay range in ms.
[[back:12]]Delete previous 12 characters.
[[raw:literal [[text]]]]Output payload as plain text.
[[note:any text]]No-op comment marker.

Escaping and Nested Syntax

  • Escape literal markers with backslash: \[[ \]].
  • Escape delimiter pipe with \|.
  • Nested directives are supported inside choice/chance/repeat.
  • Range parser supports -, –, —, ~, and to.

Complete Example

Headline draft [[rev:temporary title=>final title]]
[[pause:900-1500]]

[[choice:
The committee supports the motion.||
The committee broadly supports the motion.||
The committee supports the motion with caveats.
]]

[[chance:35|[[del:This sentence is typed and removed.]]]]
[[speed:22-34]]fast segment[[speed:default]]
[[fix:random]][[fixdelay:600-2000]]
[[repeat:2|[[choice:short echo.||slightly longer echo.]] ]]
[[back:3]]end.

快速开始

  1. 在弹窗中启用 Custom writing syntax。
  2. 使用 [[命令:参数]] 形式写语法。
  3. 点击开始后，插件按语法控制输入顺序。

注意：检测到语法指令后，插件内置的随机 typo/空白噪点/自然停顿会自动关闭，避免和脚本冲突。

核心指令

指令含义
[[rev:draft=>final]]先打草稿，停顿，删掉草稿，再打最终文本。
[[del:text]]输入后删除。
[[pause:1200]]固定停顿毫秒数。
[[pause:500-1600]]区间随机停顿。
[[choice:A||B||C]]随机选一个分支。
[[chance:35|text]]按概率执行嵌套文本。
[[repeat:3|text]]重复执行 N 次。

行为控制指令

指令含义
[[speed:20-45]]设置每 100 字用时（秒）。
[[speed:default]]恢复为当前会话基线速度。
[[typo:on/off/default/2-5]]在语法脚本内部控制 typo。
[[ws:on/off/default/1-3/keep/drop]]控制空白噪点策略。
[[fix:off|immediate|delayed|random|default]]控制 typo/ws 自动修正策略。
[[fixdelay:300-1800]]设置修正延迟区间（毫秒）。
[[back:12]]删除前 12 个字符。
[[raw:literal [[text]]]]按原样输出，不当作指令。
[[note:any text]]注释占位，不执行。

转义与嵌套

  • 字面量标记转义：\[[ \]]。
  • 竖线分隔符转义：\|。
  • choice/chance/repeat 支持嵌套指令。
  • 区间解析支持 -、–、—、~、to。

完整示例

Headline draft [[rev:temporary title=>final title]]
[[pause:900-1500]]

[[choice:
The committee supports the motion.||
The committee broadly supports the motion.||
The committee supports the motion with caveats.
]]

[[chance:35|[[del:This sentence is typed and removed.]]]]
[[speed:22-34]]fast segment[[speed:default]]
[[fix:random]][[fixdelay:600-2000]]
[[repeat:2|[[choice:short echo.||slightly longer echo.]] ]]
[[back:3]]end.
